==============================
日本語 (ja)
==============================

このアップデートでは、データのインポート/エクスポート機能の不具合を修正しました。

・バックアップの復元（インポート）が途中で失敗することがある問題を修正
・同じバックアップを再インポートすると写真が重複して追加される問題を修正
・「置換」を選択しても既存の写真が正しく置き換わらない問題を修正
・ファイルAppでバックアップファイル（.bodylapse）が選択できないことがある問題を修正
・iCloud Drive上のバックアップを、インポート前に自動でダウンロードするよう改善
・動画を含むバックアップに対応するため、最大サイズを2GBに拡大
・全体的な安定性を向上

いつもBodyLapseをご利用いただきありがとうございます。

==============================
English (en-US)
==============================

This update fixes issues with data import and export.

• Fixed an issue where restoring a backup could fail partway through
• Fixed duplicate photos being added when importing the same backup more than once
• Fixed the "Replace" option not actually replacing existing photos
• Fixed backup files (.bodylapse) sometimes being unselectable in the Files app
• Backups stored in iCloud Drive are now downloaded automatically before importing
• Increased the maximum backup size to 2 GB to support backups that include videos
• Improved overall stability

Thank you for using BodyLapse!

Español (es-ES)
==============================

Esta actualización corrige problemas con la importación y exportación de datos.

• Se corrigió un problema por el que la restauración de una copia de seguridad podía fallar a mitad del proceso
• Se corrigió la duplicación de fotos al importar la misma copia de seguridad varias veces
• Se corrigió la opción «Reemplazar», que no reemplazaba realmente las fotos existentes
• Se corrigió un problema por el que los archivos de copia de seguridad (.bodylapse) no se podían seleccionar en la app Archivos
• Las copias de seguridad guardadas en iCloud Drive ahora se descargan automáticamente antes de importarse
• Se aumentó el tamaño máximo de la copia de seguridad a 2 GB para admitir copias que incluyen vídeos
• Mejoras generales de estabilidad

¡Gracias por usar BodyLapse!

• iCloud Drive에 저장된 백업을 가져오기 전에 자동으로 다운로드하도록 개선
• 동영상이 포함된 백업을 지원하기 위해 최대 백업 크기를 2GB로 확대
• 전반적인 안정성 개선
